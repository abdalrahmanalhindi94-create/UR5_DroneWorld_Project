# UR5 Drone World Simulation Project

## Overview
This project contains the code and simulation world used to control a Universal Robots UR5 robotic arm inside a custom Gazebo drone race track world using ROS 2 Jazzy and Gazebo Harmonic.

## Contents
- code/ur_back_and_forth/: Python ROS 2 nodes for controlling the UR5 robot
- gazebo_world/: Custom Gazebo world file with shifted drone gates
- video/: Demonstration videos
- presentation/: Internship presentation slides
- documents/: Reports and notes
